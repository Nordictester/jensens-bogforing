# Jensens Bogføring

Statisk hjemmeside til [Jensens Bogføring](https://jensensbogforing.dk/).

Åbn `index.html` lokalt, eller se den live på GitHub Pages.

## Rediger priser (ejer)

1. Klik på **Admin** nederst til højre på siden.
2. Log ind med koden (standard: `Sondermarken23` — kan skiftes i `assets/js/admin.js`).
3. Opret en GitHub-nøgle med skriveadgang til repoet — se hjælp på adminsiden.
4. Ret priserne og gem.

Priserne ligger i `assets/data/prices.json` og vises automatisk på `priser.html`.
